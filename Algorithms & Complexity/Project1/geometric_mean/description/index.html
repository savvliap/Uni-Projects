Εύρεση γεωμετρικού μέσου από πίνακα ακεραίων.

Σύνοψη

Σε αυτή την άσκηση πρέπει να γράψετε κώδικα που θα επιστρέφει τον γεωμετρικό μέσο ενός πίνακα με ακεραίους.

Περιγραφή

Στο πρόγραμμα θα συμπληρώσετε την μέθοδο findGeometricMean στην κλάση FindGeometricMean στο πακέτο gr.duth.ee.euclid.datastructures.geometric_mean.
Η μέθοδος πρέπει να επιστρέφει έναν δεκαδικό (double) με τιμή ίση με τον γεωμετρικό μέσο του πίνακα που δίνεται ως όρισμα.
Ο πίνακας που δίνεται ως όρισμα θα έχει πάντα μέγεθος μεγαλύτερο ή ίσο του 1 και δεν χρειάζεται να ελέγξετε περιπτώσεις όπου το μέγεθος είναι 0 ή το όρισμα null. Επιπλέον δεν θα υπάρχουν μηδενικά στον πίνακα.

Είσοδος

Ένας μη κενός πίνακας ακεραίων (int[]) με τιμές στοιχείων στο κλειστό διάστημα [1, 100].

Έξοδος

Επιστροφή του γεωμετρικού μέσου.

Παράδειγμα

int[] values = new int[]{1, 2, 5};
int foundGeometricMean = gr.duth.ee.euclid.datastructures.geometric_mean.FindGeometricMean.findGeometricMean(values);
assert(foundGeometricMean == 2.1544);
